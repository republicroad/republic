
### toast

toast.promise 可以接受一个 promise (future) 对象, 如果操作成功，则进行 success 回调.
如果运行出错，则会以 toast 提示条提醒出错.

```ts
async function onSubmit(data: z.infer<typeof formSchema>) {
	setIsLoading(true)
	toast.promise(axios.post("/api/email_otp", data), {
	loading: 'Sending email...',
	success: () => {
		setIsLoading(false)
		form.reset()
		navigate({ to: '/otp' })
		return `Email sent to ${data.email}`
		},
	error: (err) => {
		setIsLoading(false)
		// Access custom error message from server response
		return err.response?.data?.message || err;
		},
	})
}
```
