### compiler and linker

[Address space layout randomization](https://en.wikipedia.org/wiki/Address_space_layout_randomization)  

python3.8 ubuntu
```bash
sudo apt install build-essential zlib1g-dev libncurses5-dev libgdbm-dev libnss3-dev libssl-dev libreadline-dev libffi-dev libsqlite3-dev wget libbz2-dev 

# python3.8 openssl1.1.1 编译: 先编译 openssl 到 /opt/openssl 到非默认系统搜索路径: 
make distclean 
sudo ./config -fPIC -shared --prefix=/opt/openssl -Wl,-rpath=/opt/openssl/lib 
make -j4 
sudo make install 
# 卸载 
sudo make uninstall 
# 再编译python3.8: 
./configure --with-openssl=/opt/openssl/ CFLAGS="-I/opt/openssl/include" LDFLAGS="-L/opt/openssl/lib/ -Wl,-rpath=/opt/openssl/lib/" make -j4 make altinstall
```
