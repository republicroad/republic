
### quickjs with third package bundle

google search: **quickjs bundle big.js with vite in python code**

To bundle `big.js` with Vite and execute it using QuickJS in Python, use Vite to create a single-file ESM bundle, then load and run this file using `quickjs-python`. This approach allows Python to access high-precision arithmetic library `big.js` without relying on a browser or Node.js runtime.

1. Set Up the JS Project (Vite + big.js)

First, initialize a frontend project to bundle `big.js`.

bash

```
mkdir js-bundle && cd js-bundle
npm init -y
npm install big.js
npm install -D vite
```

Create an entry point `main.js`: [[1](https://til.simonwillison.net/npm/self-hosted-quickjs#:~:text=You%20can%20use%20QuickJS%20compiled%20to%20WebAssembly,You%20can%20load%20the%20bundle%20using%20a)]

javascript

```
import Big from 'big.js';

// Expose the function to the global scope for quickjs
window.calculate = (a, b) => {
  return new Big(a).plus(b).toString();
};
```

Configure `vite.config.js` to create a single library file:

javascript

```
export default {
  build: {
    lib: {
      entry: 'main.js',
      name: 'Calculator',
      fileName: 'bundle',
      formats: ['iife'], // IIFE is best for QuickJS
    },
  },
};
```

Run the build:

bash

```
npx vite build
```

This generates `dist/bundle.iife.js`.

2. Run with Python (`quickjs-python`)

Install the Python wrapper:

bash

```
pip install quickjs-python
```

Use this Python code to execute the bundled `big.js` logic:

python

```
import quickjs

# 1. Load the bundled file generated by Vite
with open("js-bundle/dist/bundle.iife.js", "r") as f:
    bundle_code = f.read()

# 2. Initialize QuickJS context
context = quickjs.Context()

# 3. Evaluate the bundle (injects the code)
context.eval(bundle_code)

# 4. Use the exposed 'calculate' function
result = context.eval("window.calculate('0.1', '0.2')")
print(result)  # Output: 0.3
```

Key Takeaways

- **Vite Configuration:** Using `formats: ['iife']` in `vite.config.js` bundles all dependencies into one executable script without needing module loaders. [[1](https://github.com/vitejs/vite/discussions/11034#:~:text=on%20Nov%2022%2C%202022,-Original%20comment%20in&text=Hello%20dudes%2C,Thanks%20a%20lot!&text=vite%20is%20build%20by%20rollup,manualChunks%20configuration%20of%20rollup%20.)]

- **QuickJS Execution:** `quickjs-python` acts as a bridge, running the bundled JS code directly within Python's memory space. [[1](https://github.com/quickjs-zh/quickjs-python#:~:text=Usage&text=Simple%20types%20like%20int%2C%20floats,%2C%20please%20see%20test_quickjs.py%20.)]

- **Performance:** This is much faster and more lightweight than spinning up a Node.js process.




### vitejs with host lang UDF

[https://pypi.org/project/quickjs/](https://pypi.org/project/quickjs/)

```
import quickjs

# 1. Define the Python function
def python_add(a, b):
    return a + b

# 2. Create a QuickJS context
ctx = quickjs.Context()

# 3. Register the function
ctx.add_callable("add", python_add)

# 4. Use it in JavaScript
result = ctx.eval("add(2, 3)")
print(result)  # Output: 5
```
